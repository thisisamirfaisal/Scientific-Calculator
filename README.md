# 🧮 Scientific Calculator

A fully-featured, dark-themed scientific calculator built with **Streamlit**.

## Features

- **Basic arithmetic** — addition, subtraction, multiplication, division, modulo, power
- **Scientific functions** — sin, cos, tan (+ inverses), hyperbolic functions, log, ln, log₂, √, ∛, ⁿ√, exp, factorial, abs, floor, ceil
- **Constants** — π, e, φ (golden ratio), τ, ∞
- **Angle mode** — toggle between degrees and radians
- **Memory** — M+, M−, MR, MC
- **Expression history** — last 10 calculations
- **Error handling** — division by zero, domain errors, overflow
- **Polished dark UI** — JetBrains Mono display, custom CSS, badge indicators

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```
